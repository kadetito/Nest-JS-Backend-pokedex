## Ejecutar en desarrollo


1. Clonar repositorio
2. Ejecutar
```
yarn install

```
3. Tener el CLI de Nest instalado
```
npm i -g @nestjs/cli

```
4. Levantar BBDD
```
docker-compose up -d

```

* MongoDB
